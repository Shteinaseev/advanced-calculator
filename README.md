# Advanced Calculator

Реализация алгоритма вычисления выражений через ОПЗ
(стековый калькулятор)
• Чистый JavaScript без библиотек
• Адаптивный интерфейс, валидация ввода, обработка ошибок

## 📁 Структура проекта

- `index.html` — интерфейс калькулятора
- `style.css` — стили
- `script.js` — логика калькулятора
